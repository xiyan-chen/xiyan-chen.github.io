My San Guo Sha Dealing Simulator!

(Even with a user interface :D)

I am thinking what to write here

In San Guo Sha (Legends of the Three Kingdoms, National War Mode), each player receives eight generals from four factions (Wei, Shu, Wu, and Qun). Each player must select two generals from the same faction.
I (was advised to) write a Python dealing program to simulate the probability that two players in the game have no common faction to select from their eight-general pool.
I ran up to 10^9 dealing trials:
If the number of generals in each faction is 100 (which approximates sampling without replacement), the probability is about 5.6%.
If the number of generals in each faction is 20 (which approximates sampling with replacement), the probability is about 6.3% — slightly higher, which makes perfect sense!
Credit goes to Bretonia’s Shy Ghost, who also wrote a Python program to verify my result!
(And who believes my computer would crash if I actually tried 10^9 dealing trials.)
